## 三、UI优化（保持现有布局）

### 3.1 落地页文案优化

**第一屏**：
```
接单5分钟，猜他想要什么花2小时。
这活儿你还没干够？

灵思工作台 —— 帮你把客户混乱的需求，理成清楚的单子。

三颗定心丸：
· 语音放着听一遍，要点自动记下来
· 手绘草图传上来，整理成确认单
· 客户确认过的东西，自动拆成待办任务

[别猜了，接一单试试]
```

**第二屏**：
```
每天少猜1小时，一年多出30天假期。
这笔账，算得过来吧？

功能对照表：
听三遍语音，记一堆乱字 → 放着听一遍，要点自动整理好
做一版稿，客户说完全不对 → 先发确认单，让他点头再动手
群里翻了半天找不到那句话 → 每一条需求都钉在项目里
```

**第三屏**：
```
老板突然问"最近都干了什么"，
别人翻箱倒柜，你一键打印。

做新媒体的：公众号文章写好了？灵思帮你检查标题、段落、格式。
做品宣的：活动方案、品牌策略，一键生成，格式规范。
```

**第四屏**：
```
知道你在担心什么，我们摊开说。

Q: 是不是又要下个巨大软件？
A: 不用。浏览器打开就能用，有网就行。

Q: 我客户资料放网上安全吗？
A: 随时一键打包下载到电脑里。你的就是你的，我们不留。

Q: 太高级了学不会，怕吃灰。
A: 会拖文件、会打字，当天就能上手。学不会你骂我。
```

**第五屏**：
```
我们也接过语音方阵，也改过八遍稿。
所以，灵思是为我们自己做的。

别猜了。让灵思去整理，你去搞定客户。
[进入灵思工作台]
```

### 3.2 配色要求
- 主色调：紫色 #7c3aed
- 保持现有布局不变
- 只调整文案内容
