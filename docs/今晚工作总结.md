- 核心是"做海报"、"做视频"、"写文案"
- 这些是独立的工具入口

- **核心**：全案策划工作台

📋 客户背景 → 🎯 需求分析 → 🔍 竞品分析 → 💡 创意策略 → 📦 执行方案
