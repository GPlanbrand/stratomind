## 2. 核心功能使用

### 2.1 做一张海报 🖼️

用AI帮您快速生成创意海报！

**操作步骤**

**第一步：选择海报类型**
1. 点击「做一张海报」入口
2. 选择海报类型：
   - 产品宣传海报
   - 活动海报
   - 节日祝福海报
   - 社交媒体封面
   - 其他自定义海报

**第二步：输入需求**
1. 输入**产品/主题名称**
2. 选择**海报风格**（科技风、简约风、复古风、活泼风等）
3. 选择**海报尺寸**（手机壁纸、朋友圈封面、宣传单、易拉宝等）
4. 填写**其他需求**（如品牌色、slogan、必含元素等）

**第三步：获得创意方案**
1. 点击「开始生成」按钮
2. 等待AI创意生成（约10-30秒）
3. 查看生成的海报方案
4. 如需调整，可点击「重新生成」或「微调」

> 💡 **小技巧**：需求描述越详细，生成效果越好！例如："帮我生成一张护肤品宣传海报，主打补水功效，目标用户是20-30岁年轻女性"

---

### 2.2 做一套方案 📋

AI帮您生成专业的营销方案！

**操作步骤**

**第一步：选择方案类型**
1. 点击「做一套方案」入口
2. 选择方案类型：
   - 营销策划方案
   - 活动策划方案
   - 品牌规划方案
   - 产品推广方案
   - 社交媒体运营方案

**第二步：填写项目信息**
1. 输入**项目名称**
2. 选择**行业领域**（科技、电商、教育、金融、餐饮等）
3. 填写**目标受众**（年龄、性别、消费习惯等）
4. 输入**预算范围**（选填）
5. 填写**核心诉求**（您希望方案达成的目标）
6. 添加**其他补充信息**

**第三步：生成完整方案**
1. 点击「生成方案」按钮
2. 等待AI深度分析并生成（约30秒-2分钟）
3. 查看生成的完整方案，包含：
   - 方案概述
   - 市场分析
   - 创意策略
   - 执行计划
   - 预算分配
4. 可下载、编辑或重新生成

> 💡 **小技巧**：完整的项目信息能帮助AI生成更贴合实际需求的方案哦~

---

### 2.3 做一条视频 🎬

AI帮您快速生成视频创意脚本！

**操作步骤**

**第一步：选择视频类型**
1. 点击「做一条视频」入口
2. 选择视频类型：
   - 产品介绍视频
   - 品牌宣传视频
   - 活动回顾视频
   - 短视频种草
   - 培训教学视频

**第二步：输入创意点**
1. 输入**视频主题**
2. 选择**视频时长**（15秒、30秒、1分钟、3分钟等）
3. 填写**核心卖点**（必含的信息点）
4. 选择**视频风格**（活泼、正式、感人、科技感等）
5. 输入**目标平台**（抖音、小红书、B站、微信视频号等）

**第三步：生成脚本**
1. 点击「生成脚本」按钮
2. 等待AI创意生成（约10-30秒）
3. 查看生成的脚本，包含：
   - 分镜设计
   - 画面描述
   - 配音台词
   - 背景音乐建议
   - 特效建议
4. 可导出脚本或重新生成

> 💡 **小技巧**：不同平台的视频风格差异很大，填写目标平台能让AI生成更合适的脚本！

---

### 2.4 写一篇文案 ✍️

AI帮您创作各类文案！

**两种模式**

**模式一：写新文案**
1. 点击「写一篇文案」入口
2. 选择文案类型：
   - 产品卖点文案
   - 品牌故事
   - 社交媒体帖子
   - 公众号文章
   - 广告语/slogan
   - 直播话术
3. 输入文案需求和关键信息
4. 点击「生成文案」
5. 查看并编辑生成的内容

**模式二：审核现有文案**
1. 在左侧导航点击「写一篇文案」
2. 选择「文案审核」模式
3. 粘贴您现有的文案内容
4. 选择审核维度（语法、品牌调性、转化效果等）
5. 点击「开始审核」
6. 查看AI提供的修改建议

> 💡 **小技巧**：好的文案需要反复打磨，建议结合AI建议和您的实际经验进行调整~

## 6. 常见问题FAQ

### Q1：如何修改已生成的方案？

**方法一：重新生成**
1. 打开需要修改的项目
2. 点击「重新生成」按钮
3. 调整输入条件
4. 生成新版本

**方法二：在线编辑**
1. 打开项目详情页
2. 点击「编辑」进入编辑模式
3. 直接修改内容
4. 点击「保存」

**方法三：版本回退**
1. 打开项目详情页
2. 点击「版本历史」
3. 选择需要的旧版本
4. 点击「恢复到该版本」

---

### Q2：如何导出文件？

**导出步骤**
1. 打开需要导出的项目
2. 点击「导出」或「下载」按钮
3. 选择导出格式：
   - PDF：适合打印和正式文档
   - Word (.docx)：方便二次编辑
   - 图片 (.jpg/.png)：海报类项目
   - 纯文本 (.txt)：仅文字内容
4. 点击「确认导出」
5. 等待文件生成
6. 点击「下载」保存到本地

**批量导出**
1. 进入「我的项目」
2. 勾选需要导出的多个项目
3. 点击「批量导出」
4. 选择导出格式
5. 等待打包完成
6. 下载ZIP压缩包

---
