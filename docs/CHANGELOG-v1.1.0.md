## 🎯 核心升级：体验层微调

本次更新重点优化下沉市场用户体验，通过"无声的降维打击"提升用户感知。

### 1. 默认字号调整为15px
- **调整前**: 13-14px
- **调整后**: 15px
- **影响范围**: 全局字体大小
- **用户收益**: "字大一点看着不累，显得这软件实在"

### 2. AI称呼去精英化
| 原称呼 | 新称呼 | 位置 |
|--------|--------|------|
| AI助手 / 文案助手 | 📝 写作秘书 | 侧边栏、浮动助手 |
| Copilot | 格式助手 | 页面标题 |

- **文案示例**: 
  - "你好！我是你的写作秘书📝"
  - "我可以帮你：写文案、想标题、润色内容"

### 3. 图标下方强制显示文字标签
- **新增功能**: 舒适模式（默认启用）
  - 全局字号 +2px（15px → 17px）
  - 图标下方显示中文标签
  - 按钮高度增加到 48px
- **用户收益**: 消除"看不懂图标意思"的认知摩擦

### 4. 下载到本地备份入口
- **位置**: /projects 首页醒目按钮
- **功能**: 一键导出项目数据为 JSON 文件
- **文件名**: `灵思项目备份_2024-XX-XX.json`
- **用户收益**: 满足"东西要攥在自己手里"的数据主权心理

## 🔧 新增功能组件

### 1. 舒适模式开关
- **位置**: 个人设置 → 舒适模式
- **功能**:
  - 全局字号切换（标准15px / 大号17px）
  - 图标文字标签开关
  - 按钮尺寸切换（标准44px / 大号48px）
- **默认状态**: 新用户默认启用

### 2. 话术包装器
- **位置**: /projects/talk-wrapper
- **功能**: 
  - 上传设计稿（JPG/PNG）
  - 生成专业风/通俗风两版解释文案
  - 一键复制
- **适用场景**: 广告公司向客户/领导解释设计方案

### 3. 公文格式刷
- **位置**: compliance.ts 服务
- **功能**:
  - 大白话 → 标准公文格式转换
  - 支持类型: 通知、备忘录、报告、决定、公告、会议纪要
  - 智能识别会议、安全生产等场景
- **示例**:
  - 输入: "下周一我们要开个会讨论安全问题"
  - 输出: 标准的会议通知格式
