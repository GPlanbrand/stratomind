# 灵思AI创意工作台 - 首页工作台UI规划

## 一、设计概述

### 1.1 设计目标
将首页从"场景选择页"转型为"工作台清单页"，用户登录后直接看到工作台概览和项目清单，提升工作效率。

### 1.2 设计原则
- **极简主义**：减少视觉噪音，突出核心内容
- **信息密度适中**：参考飞书/扣子后台，平衡信息展示与留白
- **紫色主色调**：主色 #7c3aed，辅助渐变使用
- **卡片式布局**：圆角 16px，边框柔和阴影
- **图标统一**：全部使用 lucide-react 图标库

## 二、顶部导航栏（固定）

### 2.1 布局设计
| 元素 | 位置 | 说明 |
|------|------|------|
| Logo | 左侧 | 灵思AI Logo + 产品名称 |
| 全局搜索框 | 中间偏左 | 搜索项目/客户/文档，placeholder: "搜索项目、客户或文档..." |
| 快捷入口图标 | 右侧 | 通知铃铛(带红点)、用户头像下拉菜单 |

### 2.2 样式规范
- **导航栏高度**：64px
- **背景色**：白色 #ffffff
- **底部边框**：1px solid #f0f0f0
- **Logo**：紫色主题，28px 高度
- **搜索框**：宽度 400px，圆角 8px，浅灰背景 #f5f5f5
- **阴影**：悬停时显示 `shadow-sm`

### 2.3 交互说明
- 搜索框：输入时显示搜索建议浮层，支持历史记录
- 通知图标：点击弹出通知列表
- 用户头像：点击展开下拉菜单（个人设置、切换角色、退出登录）

## 三、顶部欢迎区

### 3.1 布局设计
采用左右两栏布局，左侧是欢迎信息，右侧是快捷操作按钮。

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│  早安，张总 👋                                              │
│  您有 3 个项目正在进行中，继续加油！                          │
│                                                            │
│                                       [+ 新建项目] [📥导入]  │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 3.2 元素说明

| 元素 | 内容 | 样式 |
|------|------|------|
| 问候语 | 动态问候（早安/午安/晚安）+ 表情 | 28px 加粗，#1f1f1f |
| 提示语 | 根据数据动态生成，如"您有X个项目进行中" | 16px，#666666 |
| 新建项目按钮 | 主操作按钮 | 紫色 #7c3aed，白色文字，右侧带 + 图标 |
| 导入按钮 | 次要操作 | 白色背景，紫色边框文字 |

### 3.3 样式规范
- **区域高度**：80px
- **左侧内边距**：24px
- **背景色**：透明（继承页面背景）
- **欢迎语行高**：1.4
- **按钮样式**：
  - 主按钮：bg-purple-600，hover:bg-purple-700，rounded-xl，padding 12px 20px
  - 次按钮：border border-purple-600，hover:bg-purple-50，rounded-xl，padding 12px 20px

### 3.4 交互说明
- 新建项目按钮：点击跳转 `/projects/workspace/new`，或弹出项目类型选择浮层
- 导入按钮：点击弹出文件选择器，支持导入 JSON 备份文件

## 四、仪表盘区域

│   12    │ │    3    │ │    8    │ │    2    │ │  环形图     │
│ ↑18%    │ │         │ │         │ │ ↑本月   │ │             │
└─────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────────┘
```

### 4.2 数据指标卡片（4张）

#### 卡片1：总项目数
| 属性 | 值 |
|------|-----|
| 图标 | Briefcase（公文包）|
| 图标背景 | 紫色渐变 #7c3aed → #8b5cf6 |
| 数字 | 项目总数 |
| 辅助信息 | 环比增长率，如 "+18%"，绿色背景 |

#### 卡片2：进行中
| 属性 | 值 |
|------|-----|
| 图标 | PlayCircle（播放圆圈）|
| 图标背景 | 蓝色渐变 #3b82f6 → #06b6d4 |
| 数字 | 进度 > 0 且 < 100 的项目数 |
| 辅助信息 | 无 |

#### 卡片3：已完成
| 属性 | 值 |
|------|-----|
| 图标 | CheckCircle（勾选圆圈）|

| 数字 | 进度 = 100 的项目数 |

#### 卡片4：本月新增

### 4.3 进度环形图（第5列）

| 属性 | 值 |
|------|-----|
| 类型 | 环形进度图 |
| 数值 | 所有项目的平均完成率 |

| 轨道颜色 | #f3f4f6 |
| 轨道宽度 | 10px |

### 4.4 样式规范

**卡片通用样式**：
- 尺寸：宽度自适应，高度 120px

- 圆角：16px
- 边框：1px solid #f0f0f0
- 内边距：20px
- 阴影：hover 时显示 `shadow-md`

**图标容器**：
- 尺寸：44px × 44px
- 圆角：12px
- 位置：右上角

**数字样式**：
- 字号：32px
- 字重：700
- 颜色：#1f1f1f

**辅助文字**：

## 五、快捷数据区

横向滚动容器，容纳最近项目和待处理任务的快捷入口。

```
┌───────────────┐ ┌───────────────┐ ┌───────────────┐ ┌───────────────┐
│ 📁 项目A      │ │ 📁 项目B      │ │ ⏰ 需求待确认  │ │ 📋 策略待审核  │

└───────────────┘ └───────────────┘ └───────────────┘ └───────────────┘

### 5.2 元素说明

#### 最近项目卡片（最多3张）
| 元素 | 说明 |
|------|------|
| 项目图标 | 文件夹图标，状态对应颜色 |
| 项目名称 | 单行截断，最多12个字符 |
| 状态+时间 | "进行中 · 2小时前" |

#### 待处理任务卡片（最多2张）
| 元素 | 说明 |

### 5.3 样式规范

**快捷卡片**：
- 最小宽度：200px
- 最大宽度：280px
- 高度：100px
- 背景：白色
- 圆角：12px
- 边框：1px solid #f0f0f0
- 内边距：16px

**布局**：
- 使用 CSS Grid 或 Flexbox

- 卡片间距：16px

### 9.2 响应式布局调整

**项目卡片列表**：
- 桌面：单列（卡片宽度 100%）
