# 灵思工作台智能体升级规划方案

> 文档版本：v1.0  
> 创建日期：2025年  
> 目标：将灵思工作台升级为具备独立身份、记忆系统、自主执行能力的真正AI智能体

---

## 一、升级目标

### 1.1 总体愿景

将灵思工作台从「被动响应的对话工具」升级为「主动服务的AI智能体」，具备：

| 能力维度 | 当前状态 | 升级后 |
|---------|---------|--------|
| 独立身份 | ❌ 无邮箱/账号 | ✅ 专属邮箱 + Profile |
| 记忆系统 | ⚠️ 基础会话记忆 | ✅ 持久化跨会话记忆 |
| 自主执行 | ❌ 无法定时任务 | ✅ 定时任务 + 队列调度 |
| 主动触达 | ❌ 等待用户 | ✅ 微信/飞书/邮件推送 |

### 1.2 分阶段目标

```
Phase 1 (基础)：独立身份 + 记忆系统
Phase 2 (执行)：日历调度 + 定时任务
Phase 3 (触达)：多渠道推送 + 主动服务
```

```sql
-- 用户记忆表
CREATE TABLE agent_memory (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id VARCHAR(64) NOT NULL,
    memory_type ENUM('profile', 'preference', 'project', 'skill') NOT NULL,
    memory_key VARCHAR(128) NOT NULL,
    memory_value JSON NOT NULL,
    confidence FLOAT DEFAULT 1.0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_user_type (user_id, memory_type),
    INDEX idx_key (memory_key)
);

-- 对话历史表
CREATE TABLE conversation_history (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id VARCHAR(64) NOT NULL,
    session_id VARCHAR(64) NOT NULL,
    summary TEXT,
    vector_id VARCHAR(128),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_user (user_id),
    INDEX idx_session (session_id)
);
```

```
└─ 同一内容去重：1小时内不重复推送
```

### 5.4 技术实现

```typescript
// 推送消息格式
interface PushMessage {
  channel: 'wechat' | 'lark' | 'email' | 'sms';
  recipient: string;
  template: string;
  data: {
    title: string;
    content: string;
    actionUrl?: string;
    priority: 'high' | 'normal' | 'low';
  };
}

// 推送服务示例
class PushService {
  async send(msg: PushMessage): Promise<PushResult> {
    // 1. 检查推送频率
    if (await this.isRateLimited(msg.recipient)) {
      return { success: false, reason: 'rate_limited' };
    }
    
    // 2. 根据渠道发送
    const sender = this.getSender(msg.channel);
    const result = await sender.send(msg);
    
    // 3. 记录推送日志
    await this.logPush(msg, result);
    
    return result;
  }
}
```
